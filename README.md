# SkipQ — Smart Queue Management System

A web-based smart queue management system that allows customers to join queues remotely and track their position in real time — no app download or hardware required.

## 🔗 Live Demo
https://skipq-dc1ac.web.app

## Features
- User registration and login (Email/Password & Google Sign-In)
- Real-time queue position tracking
- Browser notifications when your turn is near
- Admin dashboard with live queue management
- Analytics dashboard with daily statistics and hourly activity charts

## Tech Stack
- Frontend: HTML, CSS, JavaScript
- Backend: Firebase Authentication & Firestore
- Hosting: Firebase Hosting

## How to Use
1. Visit the live site
2. Register or login with Google
3. Select a counter and join the queue
4. Track your position in real time
